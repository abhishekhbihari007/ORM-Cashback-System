# Download Brand Logos

To get real brand logos, download them from these sources and save them in this folder:

## Download Instructions:

1. **Amazon**: 
   - Visit: https://brandfetch.com/amazon.com
   - Download the logo (PNG or SVG format)
   - Save as: `amazon.png` or `amazon.svg`

2. **Flipkart**: 
   - Visit: https://seeklogo.com/free-vector-logos/flipkart
   - Download the logo
   - Save as: `flipkart.png` or `flipkart.svg`

4. **Meesho**:

7. **Noon**:

8. **Sephora**: 
   - Visit: https://worldvectorlogo.com/logo/sephora-logo

## File Format:
- PNG format is recommended (transparent background preferred)
- SVG format also works
- Recommended size: 200-400px width
- Make sure the file names match exactly (lowercase)
